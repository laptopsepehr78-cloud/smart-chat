# راهنمای نصب Smart Chat

## پیش‌نیازها

- وردپرس 6.5 یا بالاتر
- PHP 7.4 یا بالاتر
- دسترسی ادمین به وردپرس

## مراحل نصب

### 1. آپلود فایل‌ها
فایل‌های افزونه را در پوشه `/wp-content/plugins/smart-chat/` کپی کنید.

### 2. فعال‌سازی
از طریق منوی 'افزونه‌ها' در وردپرس، افزونه Smart Chat را فعال کنید.

### 3. تنظیمات اولیه
به منوی 'تنظیمات > Smart Chat' بروید و تنظیمات مورد نظر را انجام دهید.

## تست افزونه

1. در پنل ادمین، بخش 'تست افزونه' را پیدا کنید
2. یک پیام تست ارسال کنید
3. پاسخ را بررسی کنید

## عیب‌یابی

### ویجت نمایش داده نمی‌شود
- بررسی کنید که افزونه فعال باشد
- تنظیمات 'فعال' را چک کنید
- کش مرورگر را پاک کنید

### خطای API
- کلید API را بررسی کنید
- آدرس Endpoint را چک کنید
- اتصال اینترنت را بررسی کنید

### مشکلات RTL
- تم سایت را بررسی کنید
- تنظیمات زبان وردپرس را چک کنید
