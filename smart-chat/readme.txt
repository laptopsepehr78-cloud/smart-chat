افزونه چت هوشمند وردپرس با رابط کاربری مینیمال و پشتیبانی کامل از فارسی/RTL

== Description ==

**Smart Chat** یک افزونه چت هوشمند برای وردپرس است که با طراحی مینیمال و مدرن، امکان چت با کاربران سایت را فراهم می‌کند.

= ویژگی‌های کلیدی =

* **طراحی مینیمال و مدرن**: رابط کاربری زیبا با انیمیشن‌های نرم
* **پشتیبانی کامل از فارسی**: تمام متن‌ها و رابط کاربری به فارسی
* **پشتیبانی از RTL**: طراحی مناسب برای زبان‌های راست به چپ
* **چندین حالت پاسخگویی**: داخلی، خارجی، یا ترکیبی
* **اتصال به API خارجی**: پشتیبانی از OpenAI، Rasa، DialogFlow و API سفارشی
* **جستجوی هوشمند**: جستجو در محتوای داخلی سایت
* **قابلیت شخصی‌سازی**: تنظیم رنگ‌ها، جایگاه، متن‌ها و غیره
* **امنیت بالا**: Rate limiting، nonce verification، sanitization کامل
* **عملکرد بهینه**: بارگذاری سبک و سریع

= حالت‌های پاسخگویی =

1. **داخلی**: جستجو و پاسخ بر اساس محتوای موجود در سایت
2. **خارجی**: اتصال به سرویس‌های چت‌بات خارجی
3. **ترکیبی**: ترکیب پاسخ‌های داخلی و خارجی با وزن‌دهی قابل تنظیم

= سرویس‌های پشتیبانی شده =

* OpenAI GPT
* Rasa
* DialogFlow
* API سفارشی
* Mock Provider (برای تست)

= تنظیمات قابل شخصی‌سازی =

* فعال/غیرفعال کردن ویجت
* انتخاب جایگاه (چهار گوشه صفحه)
* تغییر رنگ اصلی
* شخصی‌سازی متن‌های خوشامد و placeholder
* تنظیم تعداد نتایج جستجو
* فعال‌سازی/غیرفعال‌سازی لاگ‌ها
* تنظیم محدودیت درخواست

== Installation ==

1. فایل افزونه را در پوشه `/wp-content/plugins/` آپلود کنید
2. افزونه را از طریق منوی 'افزونه‌ها' در وردپرس فعال کنید
3. تنظیمات را از طریق 'تنظیمات > Smart Chat' انجام دهید
4. ویجت چت به صورت خودکار در صفحات سایت نمایش داده می‌شود

== Frequently Asked Questions ==

= آیا بدون API خارجی هم کار می‌کند؟ =

بله! افزونه در حالت داخلی بدون نیاز به API خارجی کار می‌کند و از محتوای موجود در سایت برای پاسخگویی استفاده می‌کند.

= آیا امن و مطابق GDPR است؟ =

بله! افزونه از تمام استانداردهای امنیتی وردپرس پیروی می‌کند:
* Rate limiting برای جلوگیری از سوء استفاده
* Nonce verification برای امنیت AJAX
* Sanitization و escaping کامل داده‌ها
* امکان غیرفعال کردن لاگ‌ها
* تنظیم مدت نگهداری لاگ‌ها

= آیا با تم‌های مختلف سازگار است؟ =

بله! افزونه با تمام تم‌های وردپرس سازگار است و از CSS custom properties برای شخصی‌سازی استفاده می‌کند.

= آیا از RTL پشتیبانی می‌کند؟ =

بله! افزونه به صورت کامل از RTL پشتیبانی می‌کند و برای زبان‌های فارسی، عربی و سایر زبان‌های راست به چپ بهینه شده است.

= چگونه می‌توانم API خارجی اضافه کنم؟ =

از طریق پنل ادمین > تنظیمات > Smart Chat > ارتباط با API خارجی می‌توانید:
* نوع Provider را انتخاب کنید
* کلید API را وارد کنید
* آدرس Endpoint را مشخص کنید

== Developer Notes ==

افزونه از معماری OOP و namespace استفاده می‌کند:

* `SmartChat\Loader`: کلاس اصلی بارگذاری
* `SmartChat\Admin`: مدیریت تنظیمات ادمین
* `SmartChat\REST`: API REST
* `SmartChat\Router`: مدیریت درخواست‌های AJAX
* `SmartChat\Rate_Limit`: محدودیت نرخ درخواست
* `SmartChat\Providers`: اینترفیس و پیاده‌سازی Provider ها

برای افزودن Provider جدید، کلاس Provider_Interface را پیاده‌سازی کنید.
